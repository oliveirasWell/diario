# Security Policy

## Supported Versions

We support only the latest commit on the `main` branch. Security fixes are released as needed.

## Reporting a Vulnerability

- Please DO NOT open public issues for security vulnerabilities.
- Instead, create a private security advisory on GitHub:
  - https://github.com/oliveirasWell/diario/security/advisories/new
- Alternatively, you may reach the maintainer privately via GitHub profile contact.

Provide as much detail as possible:
- Affected version/commit
- Steps to reproduce
- Impact assessment
- Suggested remediation (if any)

We will acknowledge receipt within 5 business days and keep you informed of the remediation progress.
